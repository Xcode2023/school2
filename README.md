# Project

**安装依赖:**

*npm*

```cmd
npm i
```

*yarn*

```cmd
yarn
```



**运行开发环境:**

*npm*

```cmd
npm run dev
```

*yarn*

```cmd
yarn run dev
```



**打包资源:**

*npm*

```cmd
npm run build
```

*yarn*

```cmd
yarn run build
```
